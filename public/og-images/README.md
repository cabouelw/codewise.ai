## How to Create OG Images:
1. Use tools like Canva, Figma, or Photoshop
2. Dimensions: 1200x630px (recommended by Facebook/Twitter)
3. Include tool name and brief description
4. Use brand colors and CodeWise AI logo
5. Keep text readable and centered
6. Export as JPG (optimized for web)
